### HOW TO INSTALL JAVA11 :

* Manual steps for Java11 installation:
  ------------------------------------------- 
   * UBUNTU:
     ------- 
```     
sudo apt update
sudo apt install openjdk-11-jdk 
java --version 

```
   * CENTOS:
     -------
```
sudo yum update
sudo yum install java-11-openjdk-devel
java --version

```
